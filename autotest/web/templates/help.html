Frequently Asked Questions

How accurate are the test results?

AutoTest combines automated testing with manual review guidelines. While automated tests catch many issues, manual testing is still recommended for complete accessibility validation.

What browsers are supported?

AutoTest uses Chrome and Firefox for testing. Results are applicable across modern browsers that support web standards.

Can I test password-protected pages?

Currently, AutoTest focuses on publicly accessible pages. Authentication support is planned for future releases.

How do I interpret severity levels?

Critical: Blocks access for users with disabilities
Serious: Significant barriers to accessibility
Moderate: Some users may experience difficulties
Minor: Best practice violations
