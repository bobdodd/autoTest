Getting Started

What is a project?

A project is a container for organizing related websites that you want to test for accessibility. Projects help you track improvements over time and generate comprehensive reports.

Best practices

  • Use descriptive names that identify the purpose or scope
  • Include the year or version if applicable
  • Group related websites under one project
  • Add descriptions to provide context for team members

Next steps

After creating your project, you can:

  • Add websites to test
  • Configure testing rules
  • Run accessibility scans
  • View detailed reports
